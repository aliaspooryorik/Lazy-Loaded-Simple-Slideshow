Lazy-loaded-simple-slideshow

Yes, yet another jQuery slideshow plugin! I wrote this because I wanted a simple to use slideshow which would only load the aditional images if Javascript is enabled and the DOM is ready. You can also have custom HTML as a slide.

Lazy-loaded-simple-slideshow Example

<!DOCTYPE html>
<html>
<head>
    <title>Example</title>
    <style type="text/css">
    #promo-home {width:768px; height:107px; padding:0px; margin:30px 0px 30px 0px; position:relative; border:1px solid #e5e5e5;}
    #promo-home div {position:absolute; top:0; left:0; margin:0; padding:0; width:100%; height:100%; z-index: 11;}
    #promo-home-paging a {background-color:#d0d0d0; color:white; padding:5px; margin-right:5px;}
    #promo-home-paging a.current-slide {background-color:red;}
    </style>
</head>
<body>

<div id="promo-home">
    <div><img src="http://www.crisp-ebusiness.com/images/sectionHeaders/aboutHeader.jpg" alt="Default image"></div>
</div>

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>
<script src="https://raw.github.com/aliaspooryorik/Lazy-Loaded-Simple-Slideshow/master/jquery.simpleslideshow.min.js"></script>
<script type="application/javascript">
jQuery(function($){
    var slideshowData = [
        // example of an image object with link
        {src: "http://www.crisp-ebusiness.com/images/sectionHeaders/newsHeader.jpg", alt:"Ladybird", href:"http://www.aliaspooryorik.com/"}, 
        // example of an image object with no link
        {src: "http://www.crisp-ebusiness.com/images/sectionHeaders/solutionsHeader.jpg", alt:"Ladybird"}, 
        // example using straight HTML
        '<h1>Custom Slide 2</h1><p><strong>Slide with custom HTML in it.</strong></p>'
    ];
    foo = $('#promo-home').simpleSlideshow( {
        slideshowData: slideshowData
    } );
});
</script>

</body>
</html>
